Utility-first CSS framework:

- Rapid UI development

### shadcn/ui (New York Style)

Copy-paste component library built on Radix UI:

**Philosophy**:
- Components live in your codebase (not node_modules)
- Full customization control
- Built on accessible Radix primitives
- TypeScript-first
- No vendor lock-in

**40+ Components**:
- Button, Card, Dialog, Dropdown Menu
- Form, Input, Select, Textarea
- Table, Tabs, Toast, Tooltip
- Sidebar, Navigation, Command
- Accordion, Alert, Avatar, Badge
- Checkbox, Radio, Switch, Slider
- And many more...

**Installation**:
```bash
npx shadcn@latest add button
npx shadcn@latest add dialog
npx shadcn@latest add form
```

## Styling Utilities

### class-variance-authority (CVA)

Type-safe variant management:

```typescript
import { cva } from "class-variance-authority"

const buttonVariants = cva(
  "inline-flex items-center justify-center rounded-md",
  {
    variants: {
      variant: {
        default: "bg-primary text-primary-foreground",
        destructive: "bg-destructive text-destructive-foreground",
        outline: "border border-input bg-background",
        ghost: "hover:bg-accent hover:text-accent-foreground"
      },
      size: {
        default: "h-10 px-4 py-2",
        sm: "h-9 rounded-md px-3",
        lg: "h-11 rounded-md px-8"
      }
    },
    defaultVariants: {
      variant: "default",
      size: "default"
    }
  }
)
```

### clsx + tailwind-merge

Utility for merging Tailwind classes:

```typescript
// lib/utils.ts
import { clsx, type ClassValue } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}

// Usage
<div className={cn("base-class", className)} />
```

## State Management

### React Hook Form 7.63.0

Performant form management:

**Features**:
- Minimal re-renders
- Built-in validation
- TypeScript support
- Easy to use API
- Zod integration

**Example**:
```typescript
import { useForm } from "react-hook-form"
import { zodResolver } from "@hookform/resolvers/zod"
import * as z from "zod"

const formSchema = z.object({
  email: z.string().email(),
  password: z.string().min(8)
})

function LoginForm() {
  const form = useForm<z.infer<typeof formSchema>>({
    resolver: zodResolver(formSchema)
  })

  // Form logic
}
```

### Zod 4.1.11

TypeScript-first schema validation:

```typescript
import { z } from "zod"

const userSchema = z.object({
  name: z.string().min(2).max(50),
  email: z.string().email(),
  age: z.number().min(18).optional(),
  role: z.enum(["admin", "user", "guest"])
})

type User = z.infer<typeof userSchema> // TypeScript type inferred
```

### next-themes

Theme management (dark/light mode):

```typescript
import { ThemeProvider } from "next-themes"

function App({ children }) {
  return (
    <ThemeProvider attribute="class" defaultTheme="system">
      {children}
    </ThemeProvider>
  )
}

// In components:
import { useTheme } from "next-themes"

function ThemeToggle() {
  const { theme, setTheme } = useTheme()

  return (
    <Button onClick={() => setTheme(theme === "dark" ? "light" : "dark")}>
      Toggle Theme
    </Button>
  )
}
```
